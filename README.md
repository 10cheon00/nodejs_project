# How to use?
Create virtual environment.

> sudo apt-get update   
> sudo apt-get install python-virtualenv    
> cd your/project/location/   
> virtualenv myenv   
> source myenv/bin/activate   
> pip install nodeenv   
> nodeenv -p   

Clone this repository and type below codes in console.
> pip install -r requirements.txt
